# <a href="http://vishwahm.com/">vishwahm.com</a>

This is my personal website created using <a href="https://jekyllrb.com/">Jekyll</a>. I am using hyde(now <a href="https://github.com/fongandrew/hydeout">hideout</a>) theme.

As you can see, it is hosted using <a href="https://pages.github.com/">Github Pages</a>

----------------------------------------------------------------------------

Jekyll and Hyde took care of pretty much all the things required for this website. But i have worked on bringing in some small tweeks as per my requirement.

Here is the list of the things that i changed to suit my requirement

* Adding Logo Image instead of text header(website name)
* Create a index page with transript of posts than overall post content
* Creating index pages for categories
* Showing post category and tags along with the transcript in the index page
* Integrating Discus for commenting option
* Integrating google analytics for tracking the traffic
